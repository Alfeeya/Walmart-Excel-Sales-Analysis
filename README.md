# Walmart Sales Analysis (Excel)

## Project Overview
This project analyzes Walmart retail sales data using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboards

## Key Analysis
- Total Sales, Profit, and Quantity KPIs
- Category-wise sales and profit analysis
- Top 5 states by sales
- Monthly sales trend analysis

## Dashboard
The dashboard provides a high-level view of business performance using key metrics and visualizations.

## Skills Demonstrated
- Data cleaning
- Excel formulas (SUM)
- Pivot Tables
- Business insights
